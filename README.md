### 기능 소개

1. 메인 화면에서 상품 확인 기능
2. 특정 상품에 대한 상세 정보 확인 기능
3. 장바구니 기능
4. 구매 (카카오 페이) 기능
5. 해당 상품과 관련된 정보 확인 기능
6. 로그인 기능 (네이버, 카카오 로그인)

- [ ] 메인 화면
  - 이미지 슬라이드
- [ ] 회원가입 화면 (보류)
- [x] 로그인 화면
  - 네이버, 카카오 로그인 API 사용
- [ ] 상품 상세 화면
  - 신간 상품 순위 제공
  - 장바구니 기능 제공
  - 바로 구매 기능 제공
  - 리뷰 기능 제공 (보류)
- [ ] 상품 제공자 관련 화면
  - 글, 그림, 역자, 출판사 등
- [ ] 장바구니 화면
  - 구매 기능 제공 (카카오 페이 API)
- [ ] 알림 화면
- [ ] 구매내역 화면
- [ ] 구매 성공 화면
- [ ] 구매 실패 화면
- [ ] 구매 취소 화면
- [ ] 프로필 화면
- [ ] 이용약관 화면
- [ ] 개인정보 처리방침 화면
